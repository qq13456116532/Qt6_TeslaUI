# Qt6_TeslaUI

[English Version](README-en.md)

## 项目简介
Qt6_TeslaUI 是一个使用 Qt 6 构建的用户界面项目，旨在模拟特斯拉车辆的界面。该项目展示了如何利用 Qt 框架的强大功能来创建现代化、响应式的用户界面。
